# Customer Behaviour Analytics Dashboard

## Project Overview

This project is an end-to-end customer analytics solution designed to understand customer purchasing behaviour, identify high-value customers, predict retention risks, and analyze product performance.

The project combines **Python, MySQL, and Power BI** to transform raw customer transaction data into actionable business insights.

The analysis focuses on:
- Customer segmentation
- Customer Lifetime Value (CLV)
- Churn risk identification
- Customer personas
- Product portfolio analysis
- Revenue optimization

The final output is an interactive Power BI dashboard designed for business decision-making.
